# spotify-visualization
Spotifyのデータを可視化するアプリのリニューアル版。

## やりたいこと
このアプリでは、Spotifyから取得した音声分析のデータ('BPM', '音量', 'グルーヴ感', 'アコースティック感'など)を可視化して、手動でプレイリストを作成するサポートをしたいです。
